Option Name Type Default Value Description
backdrop boolean or the string static true
  • true: 带阴影，点击阴影关闭Modal
  • false: 不带阴影
  • 'static': 带阴影，点击阴影没有反应
keyboard boolean true 按下键盘上的 Esc 时是否关闭 Modal.
show boolean true Show the modal when initialized, 值为 flase 时，
可通过 $('#myModal').modal('show'); 来触发.
